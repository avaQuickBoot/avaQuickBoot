■AvaQuickBoot
　このソフトはオンラインゲーム:Alliance of Valiant Arms(以下AVA)のログイン作業を自動化するアプリケーションです。

■動作環境
　AVAが動作するWindows
　.net framework 2.0
　InternetExplorer 32bitでAVAがログインできる状態

■使い方
　感じてください。後日作成予定

■重要な変更点
  ・ショートカットファイルがVersion 1.0.6以上のバージョンと未満のバージョンで互換性がありません。
    古いバージョンで作成したショートカットは削除し作成し直してください。
